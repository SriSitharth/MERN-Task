# MERN Task Manager

A full-stack task management app using MongoDB, Express, React, and Node.js.

## Structure

- `/client`: Frontend (React)
- `/server`: Backend (Express API)

## How to run locally

### Backend
```bash
cd server
npm install
npm run dev

### Frontend
```bash
cd client
npm install
npm start

### env
```bash
PORT=5000
MONGO_URI=your_mongodb_uri_here
JWT_SECRET=your_jwt_secret_key
